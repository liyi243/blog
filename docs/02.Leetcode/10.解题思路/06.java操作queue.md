### 数据结构Queue

特点：先到先得，先入先出

单端队列：一口进一口出（像水管）

双端队列：两口都可以进出

**复杂度**

访问：o(N)

搜索：o(N)

插入：o(1)

删除：o(1)

### java队列常用操作

接口Queue，建议用LinkedList作为Queue的数据结构

创建队列

```java
Queue<Integer> queue = new LinkedList<>();
```

添加元素:```queue.add()```

获取元素：```queue.peek();```获取即将出队的元素

删除元素:```queue.poll()``` ,删除元素并返还删除的值；remove等等

判断队列是否为空：```queue.isEmpty();``` 队列长度:```queue.size();```

边删除边遍历：

```java
while(queue.isEmpty()){
  int temp = queue.poll();
  sout(temp);
}
```
